  <action>
    Create src/components/map/DrawingHandler.tsx as a component that handles map clicks for polygon drawing during create/edit mode.

    Props interface:
    ```typescript
    interface DrawingHandlerProps {
      onDrawingComplete: (coordinates: number[][]) => void
      onCancel: () => void
      isActive: boolean
    }
    ```

    Implementation requirements:
    1. Import useMapEvents from 'react-leaflet'
    2. Import { useState, useEffect } from 'react'
    3. Use useState for points array: `const [points, setPoints] = useState<[number, number][]>([])`
    4. Use useMapEvents with:
       - click: (e) => {
           const { lat, lng } = e.latlng
           // GeoJSON uses [lng, lat] order
           const newPoint: [number, number] = [lng, lat]

           // Check if clicking near first point (complete polygon)
           if (points.length >= 2 && isNearPoint(newPoint, points[0])) {
             const closedPoints = [...points, points[0]]
             setPoints([])
             onDrawingComplete(closedPoints)
             return
           }

           setPoints([...points, newPoint])
         }
       - dblclick: () => {
           if (points.length >= 3) {
             const closedPoints = [...points, points[0]]
             setPoints([])
             onDrawingComplete(closedPoints)
           }
         }
       - keydown: (e) => {
           if (e.originalEvent.key === 'Escape') {
             setPoints([])
             onCancel()
           }
         }
    5. Helper function isNearPoint:
       ```typescript
       function isNearPoint(a: [number, number], b: [number, number], threshold = 0.001): boolean {
         return Math.sqrt(Math.pow(a[0] - b[0], 2) + Math.pow(a[1] - b[1], 2)) < threshold
       }
       ```
    6. Return null - component only attaches event listeners, doesn't render anything
    7. Use useEffect to clear points when isActive becomes false

    Key behavior:
    - Click adds polygon vertex
    - Clicking near first point closes polygon (circular completion)
    - Double-click closes polygon
    - Escape key cancels drawing
    - Returns GeoJSON coordinates in [lng, lat] order
  </action>
