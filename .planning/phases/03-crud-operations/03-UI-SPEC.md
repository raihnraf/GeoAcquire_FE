## Interaction Patterns

### Create Parcel Flow

1. User clicks "Add Parcel" button in header
2. Sidebar slides in with empty form
3. User chooses creation mode:
   - **Draw Mode**: User draws polygon on map first, then fills form
   - **Form Mode**: User fills form first, then draws polygon
4. User draws polygon on map (click to add points, double-click to close)
5. User fills form fields (owner_name, status, price_per_sqm)
6. Form validates on blur and on submit
7. User clicks "Create Parcel" button
8. Show loading state on button ("Creating...")
9. On success: Show success toast, close sidebar, refresh map
10. On error: Show error toast, keep form open with error messages

### Edit Parcel Flow

1. User clicks a parcel on map
2. Sidebar opens in view mode with parcel details
3. User clicks "Edit" button
4. Form fields become editable
5. User modifies fields or clicks "Edit Geometry"
6. For geometry edit: Map enters drawing mode, user modifies vertices
7. User clicks "Save Changes" button
8. Show loading state on button ("Saving...")
9. On success: Show success toast, return to view mode, refresh map
10. On error: Show error toast, keep form open with error messages

### Delete Parcel Flow

1. User clicks a parcel on map
2. Sidebar opens in view mode with parcel details
3. User clicks "Delete" button
4. Confirmation modal appears
5. User reads warning message
6. User clicks "Delete" to confirm
7. Show loading state on button ("Deleting...")
8. On success: Show success toast, close modal, close sidebar, refresh map
9. On error: Show error toast, keep modal open

### Drawing Mode Interactions

**Click on map:**
- Add polygon vertex at clicked location
- Draw line from previous point to new point
- Show blue circle at vertex

**Hover near first point:**
- Show "complete polygon" indicator (larger circle with check icon)
- Preview fill at 30% opacity

**Double-click:**
- Complete the polygon
- Close the shape automatically
- Exit drawing mode

**Click first point:**
- Complete the polygon
- Close the shape automatically
- Exit drawing mode

**Press Escape:**
- Cancel drawing mode
- Clear all drawn points
- Return to normal map interaction
