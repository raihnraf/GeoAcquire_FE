## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

**Exceptions:**
- Icon-only touch targets: 44px minimum (WCAG 2.5.5 compliance for mobile)
- Map control buttons: 40px (Leaflet default standard)

**Source:** 8-point scale default (standard practice)

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 14px | 600 (semibold) | 1.5 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

**Source:** Inter font standard pairing, optimized for data-dense map UI

**Type scale rationale:**
- 14px body: Optimized for information density on map interfaces
- 20px heading: Clear hierarchy without overwhelming map content
- 28px display: Page titles, major section headers
- 2 weights only: Regular (400) for body, Semibold (600) for emphasis

## Color

| Role | Value | Usage |
|------|-------|-------|

| Destructive | red-600 (#dc2626) | Delete actions, error states |

| Status | Color | Usage |
|--------|-------|-------|

| Negotiating | yellow-500 (#eab308) | In-progress parcels |
| Target | red-500 (#ef4444) | Targeted parcels |

**Accent reserved for:**
- Primary CTA buttons (Add Parcel, Import)
- Active navigation/filter states
