## Component Specifications

### 1. Layout & Spacing

**Overall Layout:**
- Full-screen map container: `h-screen w-screen overflow-hidden relative`
- Map layer: `absolute inset-0 z-0`
- Header overlay: `absolute top-0 left-0 right-0 z-10 h-16`
- Status bar overlay: `absolute bottom-0 left-0 right-0 z-10 h-14`
- Sidebar overlay: `absolute top-0 right-0 bottom-0 z-20 w-80`

**Primary Visual Focal Point:**
The map container is the primary visual anchor, occupying the full viewport (100vw × 100vh). All UI elements (header, status bar, sidebar) render as overlays above the map layer. The map remains the dominant visual element at all times.

**Header (MapHeader):**
- Height: 64px (h-16)
- Background: white with border-bottom
- Padding: 16px horizontal (px-4)
- Layout: Logo left, actions center/right
- Actions gap: 8px (gap-2)

**Status Bar (MapStatusBar):**
- Height: 56px (h-14)
- Background: white with border-top
- Padding: 16px horizontal
- Layout: Parcel count left, pagination controls right

**Sidebar (ParcelSidebar):**
- Width: 320px (w-80)
- Background: white
- Padding: 24px vertical, 16px horizontal (p-4)
- Shadow: `shadow-xl`
- Animation: Slide-in from right

### 2. Typography

**Logo/Brand:**
- Size: 20px, weight 600
- Color: slate-900 (#0f172a)

**Headings:**
- Section headings: 20px, weight 600, color slate-900
- Label headings: 14px, weight 400, color slate-700

**Body Text:**
- Size: 14px, weight 400
- Color: slate-600 (#475569)
- Line height: 1.5

**Small Text:**
- Size: 12px, weight 400
- Color: slate-500 (#64748b)

### 3. Color System

**Surface Colors:**
- Primary background: slate-50 (#f8fafc)
- Card/header background: white (#ffffff)
- Border: slate-200 (#e2e8f0)
- Divider: slate-100 (#f1f5f9)

**Interactive Colors:**
- Primary button: blue-500 (#3b82f6), hover blue-600 (#2563eb)
- Secondary button: slate-100 (#f1f5f9), hover slate-200 (#e2e8f0)
- Text link: blue-500 (#3b82f6), hover blue-700 (#1d4ed8)
- Disabled: slate-300 (#cbd5e1)

**Status Colors ( Parcel Polygons):**
- Free: green-500 (#22c55e) fill at 50% opacity, stroke at 100%
- Negotiating: yellow-500 (#eab308) fill at 50% opacity, stroke at 100%
- Target: red-500 (#ef4444) fill at 50% opacity, stroke at 100%
- Nearby (buffer): blue-500 (#3b82f6) for highlighting

### 4. Interactive States

**Button States:**
- Default: Base color with 2px padding
- Hover: Darker shade (blue-500 -> blue-600)
- Active: Scale 0.98 transform
- Disabled: slate-300 background, slate-500 text, no cursor pointer
- Focus: 2px outline blue-500, outline-offset 2px

**Parcel Polygon States:**
- Default: Stroke width 2px, fill opacity 0.5
- Hover: Stroke width 3px, fill opacity 0.7
- Selected: Stroke width 4px, blue-500 stroke color

**Sidebar States:**
- Closed: Translated off-screen (translate-x-full)
- Open: Translated in (translate-x-0)
- Transition: 300ms ease-in-out

**Loading States:**
- Spinner: 48px ring spinner, border-4, border-t-slate-900, border-slate-200
- Text: "Loading map..." below spinner
- Overlay: Full screen with slate-100/80 backdrop

### 5. Responsive Behavior

**Breakpoints:**
- Mobile: < 640px (no sidebar, full-width header)
- Tablet: 640px - 1024px (sidebar 80% width, collapsible)
- Desktop: >= 1024px (sidebar 320px fixed)

**Mobile Adaptations:**
- Header: Actions wrap to second line if needed
- Sidebar: Full-width (w-full), slide-in from bottom
- Status bar: Parcel count hidden, show only pagination

**Touch Targets:**
- Minimum 44px x 44px for all interactive elements
- Icon buttons: 40px with padding to reach 44px

## Implementation Notes

1. **Icon Library:** Install `lucide-react` before implementation:
   ```bash
   npm install lucide-react
   ```

2. **Leaflet CSS:** Must import in main entry:
   ```typescript
   import 'leaflet/dist/leaflet.css'
   ```

3. **Loading Component:** Use ring spinner pattern from RESEARCH.md code examples

4. **Empty State Component:** Use lucide-react MapIcon component for visual

5. **Sidebar Animation:** Use Tailwind transition classes:
   ```tsx
   className="fixed top-0 right-0 bottom-0 z-20 w-80 bg-white shadow-xl transition-transform duration-300 ease-in-out"
   ```

6. **Status Colors:** Import from `src/lib/utils.ts`:
   ```typescript
   import { STATUS_COLORS } from '@/lib/utils'
   ```
