  <action>
    Update vite.config.ts to copy Leaflet assets (fixes missing marker icons):

    ```typescript
    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'
    {import} { viteStaticCopy } from 'vite-plugin-static-copy'

    // https://vitejs.dev/config/
    export default defineConfig({
      plugins: [
        react(),
        viteStaticCopy({
          targets: [
            {
              src: 'node_modules/leaflet/dist/images',
              dest: 'leaflet-images'
            }
          ]
        })
      ],
      server: {
        port: 5173,
        open: true
      }
    })
    ```

    Verify tsconfig.json and tsconfig.app.json have strict mode enabled.
    In tsconfig.app.json, ensure:
    ```json
    {
      "compilerOptions": {
        "strict": true,
        "noImplicitAny": true,
        "strictNullChecks": true
      }
    }
    ```

    This is CRITICAL for GeoJSON coordinate order safety (per RESEARCH.md Pitfall 1).
  </action>
